Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
вывод программы: 2 1
в функции anotherTest переменная х объявляется в теле функции, т.е. она не возвращается из функции, её значение 
возвращается, defer отрабатывает тогда, когда значение локальной переменной уже записалось в возращаемое значение, 
поэтому, в случае с anotherTest, инкремент не влияет на то, что будет возращено из функции.

инструкция defer планирует вызов указанной функции непосредственно перед выходом из внешней функии, но аргументы 
расчитываются в момент вызова инструкции defer, более того, если в инструкции defer указан вызов нескольких функций 
"одна за другой"  func1().func2().func3(), то перед выходом из внешней функции произойдет вызов только последней 
функции, остальные будут вызваны в момент вызова инструкции defer. Если в функции несколько инструкций defer, то 
отложенные вызовы записываются в стек, т.е. первой отработает функция, записанная последней
```
